This extension provides a customizable payload generator, suitable for detecting detecting OS command injection vulnerabilities. Features include:

  • support of various syntax-dependant cases
  • configurable command and argument (different feedback channels)
  • payload marking
  • sophisticated evasive techniques (regex bypassing, badchar avoidance)
  • linux + windows support
  • configurable output encoding

More information, including the code, the manual and test cases can be found at http://github.com/ewilded/SHELLING
